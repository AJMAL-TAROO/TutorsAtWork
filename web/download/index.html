Good to know

Questions before you start?

TutorsAtWork is an account-based learning app. Access is managed by your tutor or administrator.

Do I need to install anything?

No. TutorsAtWork is available both in the browser and as installed Windows and Android apps.

Can I use TutorsAtWork on iPhone or iPad?

Yes. Use the web version in your browser. The iOS app is not required for access.

Which version should I download?

Use the web app unless you specifically want an installed app. Choose the universal Android APK for Android 7.0 or newer, or the Windows x64 installer for supported 64-bit Windows 10 and Windows 11 computers.

What if my device shows an installation warning?

Confirm that the file came from this official download page, review the prompt, and follow your device's installation instructions. Contact support if you need help.

Can I create my own account?

Student access is created and managed by a TutorsAtWork administrator. Ask your tutor for your sign-in details.
